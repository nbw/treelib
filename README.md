<h1>Treelib</h1>
<p>A collection of high-quality tree photographs for educators, students and lay persons.</p>

<p>Uses Flickr API for photo hosting.</p>

<h2>Tech Stack</h2>
<ul>
<li>Ruby (Sinatra server, ERB for templating)</li>
<li>MySQL</li>
<li>React</li>
<li>SASS</li>
<li>HTML</li>
</ul>
